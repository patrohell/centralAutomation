BMS Gestion de Edificios

Los BMS (Building Management System) son sistemas electronicos/informáticos que controlan,y automatizan los elementos mecánicos, eléctricos y tecnológicos de un edificios, tales como: climatización, iluminación, motorización de consumos (electricidad, gas y agua), sistemas de vigilancia y contraincendios, entre otros.

Sistema de control Industrial

El control industrial es la aplicación de diferentes tecnologías para controlar y monitorizar un proceso, máquina o dispositivo que habitualmente cumple funciones o tareas repetitivas, reduciendo al mínimo la intervención humana, pudiendo conseguir que una empresa lleve a cabo su producción de manera óptima y más eficiente.

Sistemas SCADA

El control de supervisión y la adquisición de datos (SCADA) es una arquitectura de sistema de control que utiliza computadoras, comunicaciones de datos en red e interfaces gráficas de usuario para la gestión de supervisión de procesos de alto nivel. Las interfaces del operador que permiten el monitoreo y la emisión de comandos de proceso se manejan a través del sistema de computadora de supervisión SCADA.
